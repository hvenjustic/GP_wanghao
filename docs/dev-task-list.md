## 2. 本次验收边界

本次只以以下五类能力作为主体闭环验收范围：

1. 用户与权限。
2. 订单管理后台。
3. 低代码配置基础能力。
4. 规则编排基础能力。
5. 日志审计与可部署运行。

## 4. 不再继续推进的事项

- 低代码表单运行时深化。
- 订单导入正式入库、去重策略和失败结果回写。
- 更复杂的规则表达式、多分支治理增强和规则平台继续深化。
- 个人密码维护、角色治理细化、用户管理增强。
- 统计看板、仓配工作台。
- 第三方平台同步、物流接口、Webhook、售后、多租户和复杂审批流。

以上事项不再作为本次必须项，也不影响主体闭环验收。
